Montserrat Light

Our main font. Use this for most text. This weight is not provided by Google fonts, so there might be a few technical issues ensureing it works for every user on every page.

Montserrat Regular

"Regular" Montserrat is bold compared to our light main font. So, use this for emphasis.

Montserrat Hairline

This is the 'hairline' version of the font. Use when you want to achieve a very light feel, but be careful about readability.
